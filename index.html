1.Unity


Unity has been around since the 8th of June, 2005, aiming to "democratize" game development by making it accessible to more developers.

It was shown at Worldwide Developers Conference 2005 by Scott Forstall on Mac OS X.

The following year, Unity was named runner-up in the Best Use of Mac OS X Graphics category in Apple Inc's Apple Design Awards.

Initially released for Mac OS X, Unity was later adding support for Microsoft Windows and Web browsers

Since its release,Unity has had multiple versions to keep up with the bugs found.

So with such a rich history, why do people no longer trust it?

Simple. The new payout plan

Now, a huge distrust in the community and Unity has formed due to how vague the price payout is

Pros:

-It has everything to let you create your game from scratch

-Great tool for prototyping everything from games, to interactive visualizations

-Allows you to import game assets and write codes

-Allows you to develop games, animations, AR, VR, UI System, Particle systems, etc.

-You only have to build it once and then you can deploy anywhere!
